10TH REQUEST

12TH REQUEST

Описание запроса:

Вывести данные о докторах с количеством пациентов, приходивших к ним на приём

14TH REQUEST

Описание запроса:

Вывести данные кабинет(а)ов, в котор(ом)ых проводилось более 2-х приёмов

select office_id, count(office_id) from appointments
